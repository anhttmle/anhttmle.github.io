# Evaluation Flow
### Manually on MVP
- Human: Ask a question
- Assistant: Provide answer and references to it
  - File Path
  - Source code
- Human: Verify answer base on references
  - Verify reference is relevant
  - Write down incorrect point.

# Evaluation Metrics
- Hit Rates
- Accuracy

# Feedback loop
- Tag on reference
- Up vote/Down vote for Assistant Answer
